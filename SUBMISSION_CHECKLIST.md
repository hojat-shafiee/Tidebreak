Collect falling items to earn points while avoiding hazards.

- Smooth combo multiplier system

- Arrow Keys / A-D: Move left and right

Open index.html in your web browser and start collecting items!

"WAVES" is represented through:
- Visual waves of falling items
- Difficulty surge mechanics every 15 seconds
- Ocean-themed color palette
- Animated wave effects in background
- Energy buildup through combo system

Technology: Vanilla JavaScript + HTML5 Canvas + CSS3
No external dependencies or libraries used.
